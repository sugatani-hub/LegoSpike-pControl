# P-Control Line Trace System for LEGO SPIKE

リアルタイムOS（RTOS）上で動作するC言語を用いた、P制御（比例制御）による高精度・高速ライントレースシステム。

## 🚀 概要
大学の自由課題において、従来のカクカクとしたON/OFF制御ではなく、数理的アプローチによる「滑らかで高速な移動」を目指して開発したロボット制御プログラムです。

## 🛠️ 技術スタック
- **開発言語:** C言語
- **プラットフォーム:** LEGO SPIKE Prime / TOPPERS/ASP（リアルタイムOS）
- **主要ライブラリ:** 組込み向けカラーセンサー・モーター制御API
- **設計手法:** 周期タスク制御（`.cfg`）、コンポーネント指向設計（`.cdl`）

## 📊 制御ロジック（P制御）
ミリ秒単位の周期タスク内で以下の演算を高速ループさせ、センサーのズレ（偏差）に応じた滑らかな旋回を実現しています。

$$e = \text{target} - \text{ref}$$
$$dV = Kp \times e$$
$$VL = \text{base} + dV, \quad VR = \text{base} - dV$$

> `target = 55` / `base = 40` / `Kp = 0.5`

## 💡 成果と得られた知見
- **なめらかな高速移動の実現:** 実機検証を繰り返し、最適な比例ゲイン（`Kp = 0.5`）を導出。直線や緩やかなカーブでの移動速度を大幅に向上。
- **組込み特有の課題の理解:** 線が薄いエリアや急カーブにおいて機体がグラつく現象を実機を通して体験。

## 📂 構成ファイル
- `linetrace.c` / `linetrace.h` : P制御および走行ロジック本体
- `linetrace.cfg` : リアルタイムOSへの周期タスク静的割当て設定
- `linetrace.cdl` : コンポーネントの接続・モジュール化定義
- `Makefile.inc` : ビルド自動化スクリプト
